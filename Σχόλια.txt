Ο σέρβερ χρησιμοποιεί το πακέτο flask του pip, καθώς και το virtualenv. Δεν κατάφερα να κάνω το script
setupenv.sh να δουλέψει χωρίς sudo όσον αφορά την εγκατάσταση του virtualenv, αλλά μπορείτε να
αλλάξετε τη γραμμή 3 όπως βολεύει. Το setupenv.sh εγκαθιστά το flask σε virtual environment μέσα στο
φάκελό του, για να μη χρειάζεται να μπλέκει με το υπόλοιπο σύστημα.
Αντίστοιχα, το run_server.sh χρησιμοποιείται για την εκτέλεση του προγράμματος του server και υποθέτει
ότι χρησιμιποιείται virtual environment για τον server στο κατάλληλο directory, με εγκατεστημένο το flask.
Ο server ξεκινά στην πόρτα 5000 και είναι προσβάσιμος στο http://localhost:5000.
Το client.py είναι το πρόγραμμα που παίζει το παιχνίδι στο url που δίνεται κατά την εκτέλεση. Χρησιμοποιεί
το αρχείο miner.py για την εύρεση του bitcoin σε κάθε βήμα. Το αρχείο miner μπορεί να εκτελεστεί και
μόνο του, και σε αυτή την περίπτωση παρέχει command line interface που βοηθά έναν ανθρώπινο παίκτη
να παίξει το παιχνίδι, δεχόμενο συνεχώς έναν μαγικό αριθμό και επιστρέφοντας ένα κατάλληλο bitcoin. O client
μπορεί να εκκινήσει και με το script run_client.sh.
